# Linux Fundamentals — Week 1

## Basic Navigation
- `pwd` → shows current directory
- `cd /path` → moves to that directory
- `ls` → lists files in current directory	

## Key Directories
- `etc/` - system configuration files
- `var/log` - system logs and records

## Practiced Commands
-[x] pwd, cd, ls
-[x] cat /etc/os-release
-[x] cat /etc/hostname
-[x] cat /etc/host

# Key Files
- `/etc/hostname` - machine name on the network
- `/etc/hosts` - local DNS, resolves names before internet
